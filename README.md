# Frontend Installation
1. ติดตั้ง Node.js บนเครื่องผู้ใช้
2. รันคำสั่งเพื่อติดตั้ง dependencies ทั้งหมดที่ระบุใน package.json
```bash
  npm install
  ```
3. ติดตั้ง axios สำหรับการเรียกใช้งาน API ในฝั่ง Frontend
```bash
  npm i axios
  ```
